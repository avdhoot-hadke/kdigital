# Task Management Application

## Installation

### Backend

1. Install my-project with npm

```bash
  cd backend/
  npm install
```

2. Create .env file with to connect MongoDb and for jwt token

```bash
    MONGODB_URL=
    SECRET_KEY=
```

3. Start project

```bash
    nodemon index.js
```

### Frontend

1. Install my-project with npm

```bash
  cd frontend/
  npm install
```

2. Start project

```bash
    npm run dev
```
